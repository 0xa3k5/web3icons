# Best Practices

## Performance

**Use individual imports** for better tree-shaking:

```tsx
import { TokenBTC } from '@web3icons/react' // ✅ Good - ~2KB
import * as Web3Icons from '@web3icons/react' // ❌ Avoid - ~500KB
```

**Prefer static imports** over dynamic components when possible:

```tsx
<TokenBTC size={24} variant="branded" /> // ✅ Static - better performance
<TokenIcon symbol="btc" size={24} variant="branded" /> // ⚠️ Dynamic - use when needed
```

## Variants

**Choose variants based on context:**

- **Mono**: Text-heavy interfaces, dark mode, when colors need customization
- **Branded**: Marketing pages, portfolios, when brand colors matter
- **Background**: Cards, badges, when icons need more visual weight

```tsx
// Interface elements
<TokenBTC variant="mono" className="text-gray-600" />

// Portfolio display
<TokenBTC variant="branded" size={32} />

// Feature cards
<TokenBTC variant="background" size={48} />
```

## Styling

**Use CSS classes for common styles:**

```tsx
// Create reusable classes
<TokenBTC className="icon-hover icon-shadow" />

// CSS
.icon-hover { hover:scale-105 transition-transform; }
.icon-shadow { drop-shadow-md; }
```

## Fallbacks

**Always provide fallbacks for dynamic components:**

```tsx
<TokenIcon
  symbol={userToken}
  fallback={<div className="size-6 bg-gray-200" />}
/>

// Or string fallback for images
<TokenIcon
  symbol={userToken}
  fallback="https://example.com/default-token.png"
/>
```

## Bundle Optimization

**Monitor bundle size:**

```bash
# Check what you're importing
npx @next/bundle-analyzer

# Audit your icon usage
grep -r "from '@web3icons/react'" src/
```

**Use dynamic imports for large lists:**

```tsx
// For icon picker/gallery components
const iconComponents = await import('@web3icons/react')
```

## Dark Mode

**Design for both themes:**

```tsx
// Use mono variant with theme-aware colors
<TokenBTC
  variant="mono"
  className="text-gray-900 dark:text-white"
/>

// Or use branded variant consistently
<TokenBTC variant="branded" /> // Works in both themes
```

## API Usage

**Cache API responses:**

```typescript
const iconCache = new Map()

async function getCachedIcon(symbol: string) {
  if (iconCache.has(symbol)) {
    return iconCache.get(symbol)
  }

  const response = await fetch(`/api/icons/${symbol}`)
  const icon = await response.text()
  iconCache.set(symbol, icon)
  return icon
}
```

## Common Pitfalls

**Avoid these patterns:**

```tsx
// ❌ Don't use branded variant with custom colors
<TokenBTC variant="branded" color="#custom" />

// ❌ Don't forget fallbacks in production
<TokenIcon symbol={dynamicSymbol} /> // Missing fallback
```

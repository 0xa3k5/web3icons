# @web3icons/core

Framework-agnostic package providing direct access to SVG files. Works with vanilla JavaScript, Vue, Angular, or any web project.

## Installation

```bash
npm install @web3icons/core
```

```bash
yarn add @web3icons/core
```

```bash
bun add @web3icons/core
```

## Quick Start

### SVG Imports

```javascript
import { svgs } from '@web3icons/core'

// Access icons as data URLs
const bitcoinIcon = svgs.tokens.brandedBTC
const ethereumIcon = svgs.networks.brandedEthereum
const metamaskIcon = svgs.wallets.brandedMetamask

// Use in HTML
document.getElementById('btc-icon').src = bitcoinIcon
```

### Framework Examples

```javascript
import { svgs } from '@web3icons/core'

const img = document.createElement('img')
img.src = svgs.tokens.brandedBTC
img.width = 32
document.body.appendChild(img)
```

```vue
<template>
  <img :src="bitcoinIcon" width="32" alt="Bitcoin" />
</template>

<script>
import { svgs } from '@web3icons/core'
export default {
  data() {
    return { bitcoinIcon: svgs.tokens.brandedBTC }
  },
}
</script>
```

```typescript
import { Component } from '@angular/core'
import { svgs } from '@web3icons/core'

@Component({
  template: '<img [src]="bitcoinIcon" width="32" alt="Bitcoin">',
})
export class MyComponent {
  bitcoinIcon = svgs.tokens.brandedBTC
}
```

## Icon Access

**Data URLs** (recommended):

```javascript
import { svgs } from '@web3icons/core'
const bitcoinIcon = svgs.tokens.brandedBTC
```

**Direct file imports**:

```javascript
const btcSvg = await import('@web3icons/core/svgs/tokens/branded/BTC.svg')
img.src = btcSvg.default
```

**Metadata** (from @web3icons/common):

```javascript
import { tokens } from '@web3icons/common/metadata'
const bitcoin = tokens.find((token) => token.symbol === 'BTC')
```

## Variants

- **mono**: `svgs.tokens.monoBTC` - Monochromatic, styleable
- **branded**: `svgs.tokens.brandedBTC` - Official brand colors
- **background**: `svgs.tokens.backgroundBTC` - With circular background

## Icon Categories

- **Tokens** (2,000+): `svgs.tokens.brandedBTC`, `svgs.tokens.brandedETH`
- **Networks** (150+): `svgs.networks.brandedEthereum`, `svgs.networks.brandedBinanceSmartChain`
- **Wallets** (50+): `svgs.wallets.brandedMetamask`, `svgs.wallets.brandedCoinbase`
- **Exchanges** (40+): `svgs.exchanges.brandedUniswap`, `svgs.exchanges.brandedBinance`

## Performance

**Tree-shaking**: Import only specific icons

```javascript
import { svgs } from '@web3icons/core' // ✅ Good
import * as Web3Icons from '@web3icons/core' // ❌ Avoid
```

**Dynamic imports**: Load on demand

```javascript
async function loadIcon(symbol) {
  const svg = await import(`@web3icons/core/svgs/tokens/branded/${symbol}.svg`)
  return svg.default
}
```

## TypeScript

TypeScript support for SVG collections:

```typescript
import { svgs } from '@web3icons/core'
import type { TTokenMetadata } from '@web3icons/common'
import { tokens } from '@web3icons/common/metadata'

interface TokenDisplay {
  token: TTokenMetadata
  size: number
}

function createIcon({ token, size }: TokenDisplay) {
  const img = document.createElement('img')
  img.src = svgs.tokens[`branded${token.symbol}`]
  img.width = size
  return img
}
```

## Exports Reference

### SVG Collections

Main export providing access to all icons as data URLs:

```typescript
import { svgs } from '@web3icons/core'

interface SvgCollections {
  tokens: Record<string, string> // brandedBTC, monoBTC, backgroundBTC
  networks: Record<string, string> // brandedEthereum, monoEthereum
  wallets: Record<string, string> // brandedMetamask, monoMetamask
  exchanges: Record<string, string> // brandedUniswap, monoUniswap

```

**Usage**:

```javascript
// Token icons
const bitcoinBranded = svgs.tokens.brandedBTC
const ethereumMono = svgs.tokens.monoETH

// Network icons
const ethereum = svgs.networks.brandedEthereum

// Wallet icons
```

### Naming Conventions

#### SVG Object Keys

Icons in the `svgs` object follow this pattern:

- **Tokens**: `{variant}{SYMBOL}` → `brandedBTC`, `monoETH`, `backgroundUSDC`
- **Networks**: `{variant}{PascalCase}` → `brandedEthereum`, `monoBinanceSmartChain`
- **Wallets**: `{variant}{PascalCase}` → `brandedMetamask`, `monoCoinbaseWallet`

#### File Paths

Direct SVG imports use these paths:

- **Tokens**: `/svgs/tokens/{variant}/{SYMBOL}.svg`
- **Networks**: `/svgs/networks/{variant}/{kebab-case}.svg`
- **Wallets**: `/svgs/wallets/{variant}/{kebab-case}.svg`

### Bundle Optimization

**Tree Shaking**:

```javascript
// ✅ Efficient - only imports needed SVGs
import { svgs } from '@web3icons/core'
const { brandedBTC, brandedETH } = svgs.tokens

// ❌ Avoid - imports everything
import * as Web3Icons from '@web3icons/core'
```

**Dynamic imports**: Load on demand

```javascript
async function loadTokenIcon(symbol, variant = 'branded') {
  try {
    const module = await import(
      `@web3icons/core/svgs/tokens/${variant}/${symbol.toUpperCase()}.svg`
    )
    return module.default
  } catch (error) {
    console.warn(`Icon not found: ${symbol}`)
    return null
  }
}
```

### Advanced Examples

#### Basic Usage

```javascript
import { svgs } from '@web3icons/core'

// Create image element
const img = document.createElement('img')
img.src = svgs.tokens.brandedBTC
img.width = 32
img.height = 32
img.alt = 'Bitcoin'
document.body.appendChild(img)
```

#### Icon Fallback Handler

```javascript
import { svgs } from '@web3icons/core'

function getTokenIcon(symbol, variant = 'branded') {
  const key = `${variant}${symbol.toUpperCase()}`
  return svgs.tokens[key] || null
}

// Usage with fallback
const btcIcon = getTokenIcon('btc') || '/fallback-icon.svg'
```

# @web3icons/react

React components for Web3 Icons with TypeScript support and tree-shaking optimization.

## Installation

```bash
npm install @web3icons/react
```

```bash
yarn add @web3icons/react
```

```bash
bun add @web3icons/react
```
Requires React 18


## Quick Start

### Individual Components (Recommended)

```jsx
import { TokenBTC, NetworkEthereum, WalletMetamask } from '@web3icons/react'

function App() {
  return (
    <div>
      <TokenBTC size={32} variant="branded" />
      <NetworkEthereum size={32} variant="branded" />
      <WalletMetamask size={32} variant="branded" />
    </div>
  )
}
```

### Dynamic Components

```jsx
import { TokenIcon, NetworkIcon } from '@web3icons/react'

function DynamicExample() {
  const tokenSymbol = 'BTC'
  const networkName = 'ethereum'
  
  return (
    <div>
      <TokenIcon symbol={tokenSymbol} size={32} variant="branded" />
      <NetworkIcon network={networkName} size={32} variant="branded" />
    </div>
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| size | string \| number | 24 | Icon size in pixels |
| variant | 'mono' \| 'branded' \| 'background' | 'mono' | Visual style |
| color | string | 'currentColor' | Color (mono variant only) |
| className | string | - | CSS class name |

All icons support standard SVG props: `onClick`, `style`, `aria-label`, etc.

## Variants

**Mono**: Monochromatic, inherits text color
```jsx
<TokenBTC variant="mono" className="text-blue-500" />
```

**Branded**: Official brand colors
```jsx
<TokenBTC variant="branded" />
```

**Background**: With circular background (select icons)
```jsx
<TokenBTC variant="background" />
```

## Icon Categories

## Performance

**Tree-shaking**: Import only what you use
```jsx
import { TokenBTC } from '@web3icons/react' // ‚úÖ ~2KB
import * as Icons from '@web3icons/react' // ‚ùå ~500KB
```

**Static vs Dynamic**: Prefer individual components
```jsx
<TokenBTC size={24} /> // ‚úÖ Better performance
<TokenIcon symbol="btc" size={24} /> // ‚ö†Ô∏è Use when needed
```

## TypeScript

Full TypeScript support with type definitions:

```tsx
import { TokenIcon } from '@web3icons/react'

interface Props {
  symbol: string
  size?: number
}

function CryptoCard({ symbol, size = 24 }: Props) {
  return <TokenIcon symbol={symbol} size={size} variant="branded" />
}
```

## Component API

### Individual Icon Components

All static icon components share the same interface:

```tsx
interface IconProps extends React.SVGProps<SVGSVGElement> {
  size?: string | number          // Default: 24
  variant?: 'mono' | 'branded' | 'background'  // Default: 'mono'  
  color?: string                  // Default: 'currentColor' (mono only)
  className?: string
}
```

**Usage**:
```tsx
import { TokenBTC, NetworkEthereum, WalletMetamask } from '@web3icons/react'

<TokenBTC size={32} variant="branded" />
<NetworkEthereum size={24} variant="mono" color="#627eea" />
<WalletMetamask size={20} className="hover:scale-105" onClick={handleClick} />
```

### Dynamic Components

#### TokenIcon

```tsx
interface TokenIconProps extends IconProps {
  symbol?: string                 // Token symbol: "btc", "eth"
  address?: string               // Contract address
  network?: string               // Network for address lookup
  fallback?: React.ReactNode | string
}
```

```tsx
import { TokenIcon } from '@web3icons/react'

// By symbol
<TokenIcon symbol="btc" variant="branded" />

// By contract address  
<TokenIcon address="0xa0b86a33e6441cc8c6b01cfb5e45bcf5b92f4dc1" network="ethereum" />

// With fallback
<TokenIcon symbol="unknown" fallback={<div>?</div>} />
<TokenIcon symbol="custom" fallback="https://example.com/token.png" />
```

## Usage Examples

### Icon Grid

```tsx
const tokens = ['BTC', 'ETH', 'USDC']

function TokenGrid() {
  return (
    <div className="grid grid-cols-3 gap-4">
      {tokens.map(symbol => (
        <div key={symbol} className="flex items-center gap-2">
          <TokenIcon symbol={symbol.toLowerCase()} size={24} variant="branded" />
          <span>{symbol}</span>
        </div>
      ))}
    </div>
  )
}
```

### Wallet Selector

```tsx
const wallets = [
  { id: 'metamask', name: 'MetaMask' },
  { id: 'coinbase-wallet', name: 'Coinbase' },
  { id: 'rainbow', name: 'Rainbow' }
]

function WalletSelector({ onSelect }) {
  return (
    <div className="space-y-2">
      {wallets.map(wallet => (
        <button 
          key={wallet.id}
          onClick={() => onSelect(wallet.id)}
          className="flex items-center gap-3 p-3 border rounded hover:border-blue-500"
        >
          <WalletIcon name={wallet.id} size={24} variant="branded" />
          <span>{wallet.name}</span>
        </button>
      ))}
    </div>
  )
}
```

## Error Handling

Dynamic components handle missing icons gracefully:

```tsx
// Returns null if icon not found and no fallback
<TokenIcon symbol="invalid" />

// Uses fallback when provided
<TokenIcon symbol="invalid" fallback="?" />

// Component-level error boundary
function SafeIcon({ symbol }) {
  return (
    <ErrorBoundary fallback={<div className="w-6 h-6 bg-gray-200 rounded" />}>
      <TokenIcon symbol={symbol} variant="branded" />
    </ErrorBoundary>
  )
}
```

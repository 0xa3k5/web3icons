# Web3Icons React + Vite Example

Demonstrates `@web3icons/react` with React and Vite.

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:5173](http://localhost:5173)

## Pages

- `/` - Static imports (tree-shakable, 7 icons)
- `/dynamic` - Dynamic imports (runtime lookup)

## Bundle Analysis

```bash
bun run build:analyze
```

# Web3Icons Next.js Example

Demonstrates `@web3icons/react` with Next.js App Router.

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` - Static imports (Server Components, 7 icons)
- `/dynamic` - Dynamic imports (Client Components)

## Bundle Analysis

```bash
bun run build:analyze
```

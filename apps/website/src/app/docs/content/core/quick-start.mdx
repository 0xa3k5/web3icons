# @web3icons/core

Framework-agnostic package providing direct access to SVG files. Works with vanilla JavaScript, Vue, Angular, Svelte, or any web project.

## Installation

<InstallTabs pkg="@web3icons/core" />

## SVG Imports

Access icons as data URLs through the `svgs` object:

```javascript

// Token icons
const bitcoinIcon = svgs.tokens.brandedBTC
const ethereumIcon = svgs.tokens.monoETH

// Network icons
const polygonIcon = svgs.networks.brandedPolygon

// Wallet icons
const metamaskIcon = svgs.wallets.brandedMetamask

// Exchange icons
const uniswapIcon = svgs.exchanges.brandedUniswap
```

## Using in HTML

```javascript

// Set as image source
document.getElementById('btc-icon').src = svgs.tokens.brandedBTC

// Create dynamic elements
const img = document.createElement('img')
img.src = svgs.tokens.brandedETH
img.width = 32
img.height = 32
img.alt = 'Ethereum'
document.body.appendChild(img)
```

## Variants

All icons come in multiple visual styles:

```javascript

// Mono - Monochromatic, single color
svgs.tokens.monoBTC

// Branded - Official brand colors
svgs.tokens.brandedBTC

// Background - With circular background (tokens only)
svgs.tokens.backgroundBTC
```

## Naming Conventions

### Object Keys

Icons in the `svgs` object follow this pattern:

| Type | Pattern | Examples |
| ---- | ------- | -------- |
| Tokens | `{variant}{SYMBOL}` | `brandedBTC`, `monoETH` |
| Networks | `{variant}{PascalCase}` | `brandedEthereum`, `monoBinanceSmartChain` |
| Wallets | `{variant}{PascalCase}` | `brandedMetamask`, `monoCoinbaseWallet` |
| Exchanges | `{variant}{PascalCase}` | `brandedUniswap`, `monoBinance` |

### Direct File Paths

You can also import SVG files directly:

```javascript
// Tokens: /svgs/tokens/{variant}/{SYMBOL}.svg

// Networks: /svgs/networks/{variant}/{kebab-case}.svg

// Wallets: /svgs/wallets/{variant}/{kebab-case}.svg
```

## Dynamic Loading

Load icons on demand for better initial bundle size:

```javascript
async function loadIcon(symbol, variant = 'branded') {
  try {
    const module = await import(
      `@web3icons/core/svgs/tokens/${variant}/${symbol.toUpperCase()}.svg`
    )
    return module.default
  } catch (error) {
    console.warn(`Icon not found: ${symbol}`)
    return null
  }
}

// Usage
const btcIcon = await loadIcon('btc')
const ethIcon = await loadIcon('eth', 'mono')
```

## Helper Function

Create a utility for safe icon access with fallbacks:

```javascript

function getTokenIcon(symbol, variant = 'branded') {
  const key = `${variant}${symbol.toUpperCase()}`
  return svgs.tokens[key] || null
}

// Usage with fallback
const icon = getTokenIcon('btc') || '/fallback-icon.svg'
```

## TypeScript

Full TypeScript support included:

```typescript

interface TokenDisplay {
  token: TTokenMetadata
  size: number
}

function createIcon({ token, size }: TokenDisplay) {
  const key = `branded${token.symbol}` as keyof typeof svgs.tokens
  const img = document.createElement('img')
  img.src = svgs.tokens[key]
  img.width = size
  return img
}
```

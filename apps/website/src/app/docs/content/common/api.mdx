# API Reference

Complete reference for all @web3icons/common exports.

## Type Exports

Import types from the main entry point:

```typescript
import {
  TVariant,
  TType,
  TTokenMetadata,
  TNetworkMetadata,
  TWalletMetadata,
  TExchangeMetadata,
} from '@web3icons/common'
```

### TVariant

Icon visual style variant.

```typescript
type TVariant = 'mono' | 'branded' | 'background'
```

| Value          | Description                            |
| -------------- | -------------------------------------- |
| `'mono'`       | Monochromatic, single color            |
| `'branded'`    | Official brand colors                  |
| `'background'` | With circular background (tokens only) |

### TType

Icon category type.

```typescript
type TType = 'token' | 'network' | 'wallet' | 'exchange'
```

### TTokenMetadata

Token metadata structure.

```typescript
interface TTokenMetadata {
  id: string
  name: string
  symbol: string
  variants: TVariant[]
  addresses?: Record<string, string>
  marketCapRank?: number | null
}
```

| Property        | Type                     | Description                           |
| --------------- | ------------------------ | ------------------------------------- |
| `id`            | `string`                 | Unique identifier (e.g., `"bitcoin"`) |
| `name`          | `string`                 | Display name (e.g., `"Bitcoin"`)      |
| `symbol`        | `string`                 | Ticker symbol (e.g., `"BTC"`)         |
| `variants`      | `TVariant[]`             | Available icon variants               |
| `addresses`     | `Record<string, string>` | Contract addresses by network         |
| `marketCapRank` | `number \| null`         | Market cap ranking                    |

### TNetworkMetadata

Network metadata structure.

```typescript
interface TNetworkMetadata {
  id: string
  name: string
  shortName?: string
  chainId?: number | string
  caip2id?: string
  variants: TVariant[]
  nativeCoinId?: string
}
```

| Property       | Type               | Description                            |
| -------------- | ------------------ | -------------------------------------- |
| `id`           | `string`           | Unique identifier (e.g., `"ethereum"`) |
| `name`         | `string`           | Display name (e.g., `"Ethereum"`)      |
| `shortName`    | `string`           | Short name (e.g., `"ETH"`)             |
| `chainId`      | `number \| string` | EVM chain ID                           |
| `caip2id`      | `string`           | CAIP-2 identifier (e.g., `"eip155:1"`) |
| `variants`     | `TVariant[]`       | Available icon variants                |
| `nativeCoinId` | `string`           | ID of native coin token                |

### TWalletMetadata

Wallet metadata structure.

```typescript
interface TWalletMetadata {
  id: string
  name: string
  variants: TVariant[]
}
```

| Property   | Type         | Description                            |
| ---------- | ------------ | -------------------------------------- |
| `id`       | `string`     | Unique identifier (e.g., `"metamask"`) |
| `name`     | `string`     | Display name (e.g., `"MetaMask"`)      |
| `variants` | `TVariant[]` | Available icon variants                |

### TExchangeMetadata

Exchange metadata structure.

```typescript
interface TExchangeMetadata {
  id: string
  name: string
  type: 'cex' | 'dex'
  variants: TVariant[]
}
```

| Property   | Type             | Description                           |
| ---------- | ---------------- | ------------------------------------- |
| `id`       | `string`         | Unique identifier (e.g., `"uniswap"`) |
| `name`     | `string`         | Display name (e.g., `"Uniswap"`)      |
| `type`     | `'cex' \| 'dex'` | Exchange type                         |
| `variants` | `TVariant[]`     | Available icon variants               |

## Metadata Exports

Import metadata from the metadata subpath:

```typescript
import {
  tokens,
  networks,
  wallets,
  exchanges,
} from '@web3icons/common/metadata'
```

`tokens`

```typescript
const tokens: TTokenMetadata[]

// Example entry
{
  id: 'bitcoin',
  name: 'Bitcoin',
  symbol: 'BTC',

  addresses: {
    ethereum: '0x2260fac5e5542a773aa44fbcfedf7c193bc2c599'
```

## Utility Functions

Common patterns for working with metadata:

#### Find by ID

```typescript
const bitcoin = tokens.find((t) => t.id === 'bitcoin')
const ethereum = networks.find((n) => n.id === 'ethereum')
```

#### Find by Symbol

```typescript
const btc = tokens.find((t) => t.symbol === 'BTC')
const eth = tokens.find((t) => t.symbol === 'ETH')
```

#### Find by Chain ID

```typescript
const network = networks.find((n) => n.chainId === 137) // Polygon
```

#### Find by Contract Address

```typescript
function findTokenByAddress(address: string, network: string) {
  return tokens.find(
    (t) => t.addresses?.[network]?.toLowerCase() === address.toLowerCase(),
  )
}
```

#### Search by Name

```typescript
function searchTokens(query: string) {
  const q = query.toLowerCase()
  return tokens.filter(
    (t) =>
      t.name.toLowerCase().includes(q) || t.symbol.toLowerCase().includes(q),
  )
}
```

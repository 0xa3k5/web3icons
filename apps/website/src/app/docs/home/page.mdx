# Web3 Icons

The most complete library of cryptocurrency, blockchain network, wallet, and exchange icons.

**2,500+ icons** across multiple variants and formats.

## Quick Start

<InstallTabs pkg="@web3icons/react" />

```jsx
import { TokenBTC, NetworkEthereum, WalletMetamask } from '@web3icons/react'

function App() {
  return (
    <div>
      <TokenBTC size={32} variant="branded" />
      <NetworkEthereum size={32} variant="branded" />
      <WalletMetamask size={32} variant="branded" />
    </div>
  )
}
```

## Packages

| Package                           | Description                              |
| --------------------------------- | ---------------------------------------- |
| [@web3icons/react](/docs/react)   | React components with TypeScript support |
| [@web3icons/core](/docs/core)     | Framework-agnostic SVG files             |
| [@web3icons/common](/docs/common) | TypeScript types and metadata            |

## Icon Types

- `Tokens` (2,000+): BTC, ETH, USDC, and more
- `Networks` (150+): Ethereum, BSC, Polygon, Arbitrum
- `Wallets` (50+): MetaMask, Coinbase, Rainbow, Ledger
- `Exchanges` (40+): Uniswap, Binance, Coinbase, PancakeSwap

## Variants

- `mono`: Monochromatic, inherits text color
- `branded`: Official brand colors
- `background`: Icons with circular backgrounds

## Features

- Tree-shakeable packages
- TypeScript support
- Multiple icon variants
- Optimized SVGs
